# Topic Design（议题设计）

## 标准定义

**议题设计是将品牌私有化增长目标，重构为具有公共价值的社会化议题。**

## 详细解释

议题设计解决的核心问题是：「用户为什么愿意讨论你」。

它将品牌的商业目标，转化为公众愿意参与讨论的社会话题。一旦完成这个转化，传播就不再是品牌单方面的「推送」，而是社会自发的「接力」。

## 核心特征

1. **公共性**：议题必须与更广泛的社会群体相关，而非仅限于品牌现有用户
2. **讨论性**：议题必须具备被讨论、被争论、被表达的空间
3. **参与性**：议题必须允许公众以某种方式参与，而非仅作为旁观者

## 议题设计的四个标准

1. **足够秒懂**：用户刷到的一瞬间，就能明白发生了什么
2. **带有情绪开关**：好奇、怀疑、兴奋、意外、愤怒、站队等情绪是扩散的发动机
3. **允许参与**：适合评论、转发、下注、争论或模仿
4. **认知归品牌**：热闹过去之后，别人仍然清楚记得这件事是谁做的

## 典型案例

- 「英格兰夺冠退全款」：将促销活动转化为全民关注的社会赌局
- 「李剑怼美团」：将商家诉求转化为行业公共议题
- 「淄博烧烤」：将城市美食转化为社会情绪出口

## 在整体模型中的作用

议题设计是传播的「接口层」，它决定了社会是否愿意帮你传播。
