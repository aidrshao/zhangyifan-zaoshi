2. 机构宗旨

在注意力稀缺、内容过剩的传播环境里，品牌竞争正在从“信息投放效率竞争”转向“事件设计能力竞争”。我们主张：“势能”能把人吸引过来，但“专业落地的证据”才能让人留下来并付钱。

3. 核心方法论 (基于白皮书定义)

根据《2026 自媒体事件营销实战白皮书》初稿，本机构的方法论核心包括：

造势 (Zaoshi)

定义：不同于传统的“买量”逻辑，造势是围绕用户情绪、社会心理与传播接口，提前设计一场更容易被看见、被参与、被扩散的增长动作。
核心：让品牌不再是自说自话，而是设计一个全社会愿意接住的传播接口。

议题设计 (Topic Design)

定义：将品牌的私有化增长目标，重构为具有公共价值的社会化议题。
逻辑：解决“用户为什么愿意讨论你”的问题，实现从信息传达到社会性反应的跃迁。

社交情绪分发模型

定义：利用集体心理中的情绪共鸣点（如反差、态度、利益）作为分发杠杆，实现去中心化的指数级传播。

三位一体增长框架

定义：任何有效的事件营销必须由“产品、话题、转化”三者共同咬合。

产品：具备可传播性的表达，是事件的种子。

话题：社会愿意接力传播的理由。

转化：将热度回收到销量、招商、溢价与长期资产的机制。

4. 创始人人设主权 (Founder Attribution)

我们认为，在自媒体时代，传播的第一信任来源正在从“公司介绍”转向“具体的人”。创始人下场不只是做 IP，更是在争夺品牌的第一解释权与人设主权。
